Purdue ECE Faculty Directory
A web application for searching Purdue ECE faculty members by name or research area.

Features
Name Search: Search faculty by name with wildcard support (e.g., Dav* matches David, Dave)
Research Area Search: Find faculty by their research interests with root word matching
Responsive Design: Works seamlessly on desktop and mobile devices
How to Use
Search by Name
Select the "Search by Name" tab
Enter a faculty member's name (partial names supported)
Use wildcards () to match multiple names (e.g., "Dav" matches "David Inouye" and "Dave Janes")
Search by Research Area
Select the "Search by Research Area" tab
Enter a research keyword (e.g., "security", "machine learning")
The system will find faculty whose research areas match your query
Root word matching is supported (e.g., "security" matches "secure")
